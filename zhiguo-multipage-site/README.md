# Zhiguo Yang - Multi-page Academic Website

This package contains a more formal multi-page academic website in a faculty-homepage style.

## Included pages
- `index.html` - home
- `research.html` - research overview and future directions
- `publications.html` - featured and selected publications
- `teaching.html` - teaching philosophy, courses, mentoring
- `service.html` - selected talks, service, and academic leadership
- `style.css` - shared stylesheet

## Included PDF files
- `ZhiguoYangCV.pdf`
- `ResearchStatement-ZhiguoYang.pdf`
- `TeachingStatement-ZhiguoYang.pdf`

## Recommended next edits
1. Add your Google Scholar, ORCID, and ResearchGate links.
2. Add DOI or arXiv links for papers.
3. Add a professional headshot if you want.
4. Add a short “Openings” section if you want to recruit PhD students or postdocs.
5. If you buy a personal domain later, point it to this site.

## Fast deployment with GitHub Pages
1. Create a GitHub repository, for example `zhiguo-yang`.
2. Upload all files in this folder.
3. In the repository, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose the `main` branch and the root folder.
6. Save. Your site should be published shortly.

## Notes
- This site is intentionally plain, formal, and easy to maintain.
- All content is static HTML and CSS, so it is stable and portable.
